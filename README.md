Приложение с графическим интерфейсом на **customtkinter** для автоматической рассылки сообщений в Telegram.  
Позволяет:
- ✅ Добавлять сообщения в **отложку** (личные сообщения выбранным пользователям)
- ✅ Выполнять **рассылку по группам** (одно сообщение во все указанные группы)

## ✨ Возможности

- **Рассылка в личные сообщения** – массовая отправка сообщений выбранным пользователям (поддержка отложенной отправки).
- **Рассылка по группам** – отправка единого сообщения во все группы, указанные в списке.
- **Графический интерфейс** – удобное окно на базе `customtkinter` (современный тёмный/светлый дизайн).
- **Простая настройка** – через файл конфигурации или прямо из интерфейса.
- **Логирование** – все действия записываются для контроля.
